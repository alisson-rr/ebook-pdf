# 🧲 Guia do Atendimento Magnético

Do Atendimento Humanizado à Gestão de Alta Performance: O método C.A.R.E. e estratégias avançadas para encantar clientes, automatizar processos e maximizar resultados em 2026 e além.

## Estrutura do Projeto

```
guia-atendimento-magnetico/
├── public/
│   └── images/          # Imagens do guia
├── src/
│   ├── css/
│   │   ├── variables.css    # Variáveis CSS (cores, fontes)
│   │   ├── base.css         # Estilos base (body, tipografia)
│   │   ├── layout.css       # Layout (container, grid)
│   │   ├── components.css   # Componentes (boxes, cards)
│   │   ├── tables.css       # Tabelas
│   │   ├── responsive.css   # Media queries
│   │   └── main.css         # Arquivo principal (imports)
│   └── js/
│       └── charts.js        # Gráficos ECharts
├── index.html               # Página principal
├── package.json
└── README.md
```

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Acesse em: http://localhost:3000

## Tecnologias

- HTML5
- CSS3 (Variáveis CSS, Flexbox)
- JavaScript (ECharts para gráficos)
- Normalize.css
